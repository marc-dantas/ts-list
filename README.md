## Quick Start
You can access the application by the [github pages link](https://marc-dantas.github.io/ts-list) or build and open the project on your machine:
```console
$ # install typescript
$ npm i -g typescript
$ # compile source
$ tsc # you do this only once
$ # open the project
$ [your-browser] index.html
```
